# Sound Dropper

Async conversations

# Getting started

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Running in production

```
npm run build
node server.js
```
